Smart Scanner

The Smart Scanner is an AI-powered document scanning application developed using Python, the Pillow library, and an Optical Character Detection (OCD) algorithm. This project leverages image processing and OCR techniques to accurately detect, extract, and convert text from scanned documents into editable formats.

Key Features:

1. Optical Character Detection: Uses the OCD algorithm to recognize and extract text from scanned images.
2. Image Processing: Enhances document clarity using the Pillow library.
3. Automation: Automatically detects document edges, crops, and adjusts the document for optimal text recognition.
4. User-Friendly: Simple interface for easy scanning, text extraction, and saving of documents.

Mobility Detector

This project involves developing a Python-based traffic and mobility detection system using background subtraction and contour analysis techniques. The system aims to detect and track moving objects, such as vehicles and pedestrians, in real-time video streams.

Key Features :

1. Background Subtraction: Isolating moving objects from a static background using techniques like Gaussian Mixture Models (GMM).
2. Contour Analysis: Detecting the boundaries of moving objects for accurate tracking.
3. Real-Time Processing: Ensuring low-latency performance for practical applications.
